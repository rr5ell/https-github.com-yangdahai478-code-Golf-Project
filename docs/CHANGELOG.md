# 项目变更日志

## [2026-04-02] 初始版本提交

### 新增功能

1. **核心物理引擎** (`physics_engine.py`)
   - 基于 RK4 (四阶龙格-库塔) 方法的高精度数值积分
   - 自适应时间步长控制，避免数值不稳定
   - KDTree + 局部加权平均的地形插值
   - 完整的物理模型：重力分量、滚动摩擦、空气阻力

2. **图形界面** (`gui.py`)
   - 实时 3D 地形可视化
   - 俯视图轨迹显示
   - 交互式参数调整
   - 点云文件加载功能

3. **交互式界面 V2** (`interactive_gui_v2.py`)
   - 增强的用户交互体验
   - 更直观的推杆控制
   - 实时参数反馈

4. **演示程序** (`demo.py`)
   - 预设演示地形
   - 快速测试模拟

5. **XYZ 格式支持** (`run_with_xyz.py`)
   - 支持 XYZ 坐标格式
   - 扩展点云格式兼容性

### 技术特点

- ✅ RK4 积分方法（高精度，稳定）
- ✅ KDTree + 加权插值（快速，抗噪点）
- ✅ 自适应时间步长
- ✅ 边界检测
- ✅ 速度归零处理

### 物理常数

- 高尔夫球直径: 42.67 mm
- 高尔夫球质量: 45.93 g
- 草坪滚动摩擦系数: 0.05-0.15 (可调)
- 空气阻力系数: 0.47
